<p></p>
1)download the files from the github
<p></p>
2)open visual studio code and then open the terminal
<p></p>
3)cd server
<p></p>

<p></p>
5)npm start
<p></p>
6)open another terminal 
<p></p>
7)cd client
<p></p>
8)write this command on the terminal :npm install
<p></p>
9)npm start
